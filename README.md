# CMS
A streamlined Complaint Management System (CMS) for registering, assigning, and tracking user grievances. Built to ensure timely resolutions, improve workflow transparency, and boost overall satisfaction


### ✨ About The Project
This project provides a centralized, easy-to-use platform for handling organizational complaints. It uses a separate client-server architecture for maximum scalability. Users can securely submit grievances, while administrators get a dedicated dashboard to review, assign, and resolve tickets efficiently.

### 🛠️ Built With
The system is fully powered by the modern MERN stack. 
* **MongoDB:** Stores and queries all system data.
* **Express & Node.js:** Handle the backend APIs and application logic securely.
* **React:** Provides a fast, dynamic single-page user interface.

### 🌟 Key Features
**🔐 Secure Authentication:** Role-based access ensures standard users and administrators only see what they need to.
**📊 Real-Time Tracking:** Users can monitor the exact status of their submitted grievances at any time.
**⚙️ Admin Dashboard:** Administrators are equipped with powerful backend tools to filter, assign, and resolve incoming tickets quickly.
